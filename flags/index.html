Flag probes

8 flags · order matches the modal

Reading the results — the launch path matters. The live agentic path (preCreateSeleniumSession → BrowserFlagsHandler) applies flag args and prefs, dedupes repeats, and its defaults already allow clipboard (pref = 1). The WDIO replay path ignores user flags entirely and force-denies geolocation/camera/mic/notifications. An older pwCaps path appends args only. So the same flag can behave differently per run type — exactly what this page exposes. Confidence per card: strong (definitive JS signal) vs heuristic (inferred — verify by comparing on/off). See the launch diff for the full command-line picture.
